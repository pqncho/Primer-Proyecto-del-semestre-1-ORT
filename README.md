🏃‍♂️ Street Race Management System

A web application developed for managing street races across Uruguay.

This project was created as part of the Programming I Final Project at ORT University Uruguay, focusing on the implementation of a complete management platform using modern web technologies.

🚀 Project Overview

The goal of this application is to streamline the organization and management of street races by providing tools for registration, validation, reporting, and data visualization.

The system allows organizers to manage races, participants, sponsors, and registrations while offering useful statistics and interactive visualizations.

✨ Features
🏁 Race Management
Create and manage race events
Track participant capacity
Manage race information and details
👤 Runner Registration
Register runners and maintain participant records
Validate medical certificates before registration
Handle race enrollments
🤝 Sponsor Management
Register and manage sponsors
Associate sponsors with race events
📊 Statistics & Reports
Generate general statistics
Perform specific queries and data analysis
Display relevant race information dynamically
🗺️ Interactive Map
Visualize races across Uruguay
Display location-based information through an interactive map
📄 PDF Generation
Download registration details in PDF format
Generate printable registration documents
